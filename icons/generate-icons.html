Option 3: Use the Command Line (requires ImageMagick)

If you have ImageMagick installed, you can generate icons from the terminal:

cd icons

# Create a simple icon (512x512)
convert -size 512x512 gradient:#1a1a2e-#e94560 \
        -gravity center -pointsize 200 -fill white \
        -annotate +0+0 "T26" icon-512x512.png

# Then resize to other sizes
for size in 72 96 128 144 152 192 384; do
  convert icon-512x512.png -resize ${size}x${size} icon-${size}x${size}.png
done

Required Icon Sizes

Your PWA needs these icon sizes in the icons/ directory:

  • icon-72x72.png
  • icon-96x96.png
  • icon-128x128.png
  • icon-144x144.png
  • icon-152x152.png
  • icon-192x192.png
  • icon-384x384.png
  • icon-512x512.png

Design Guidelines

  • Keep it simple - Icons should be recognizable at small sizes
  • Use high contrast - Text/symbols should stand out from background
  • Safe zone - Keep important elements 10% away from edges
  • Test on device - View your icon on an actual phone home screen
